# Agency Brand System

Agency should feel like a working command center: quiet enough to live in all day, sharp enough to look deliberate in a screenshot, and technical enough that developers immediately understand this is not a chatbot wrapper.

## Wordmark

```txt
    ___    ______ ______ _   __ ______ __  __
   /   |  / ____// ____// | / // ____// / / /
  / /| | / / __ / __/  /  |/ // /    / /_/ /
 / ___ |/ /_/ // /___ / /|  // /___ / __  /
/_/  |_|\____//_____//_/ |_/ \____//_/ /_/
```

Use the wordmark in terminal surfaces, setup scripts, release notes, and screenshot frames. Keep it monochrome or signal-gold; do not add gradients to terminal output.

## Voice

- Lead with verbs: build, inspect, approve, route, wake, commit.
- Keep copy operational, not mystical.
- Use "office" for the user-facing organization metaphor.
- Use "ledger" when talking about auditability and durable evidence.
- Use "transformer" only for future Mobius integration or architecture docs, not first-run setup.

## Screenshot Direction

- Dark background.
- Signal Gold wordmark.
- Relay Cyan for live process/routing status.
- Ledger Green for successful gates and approvals.
- Show real commands and evidence, not marketing filler.
